# Kebijakan Privasi WaterVault

**Terakhir diperbarui: 21 Juli 2026**

Terima kasih telah menggunakan WaterVault. Kebijakan privasi ini menjelaskan cara kami menangani informasi Anda.

## Pengumpulan dan penggunaan informasi

Aplikasi ini **tidak mengumpulkan atau mengunggah data pribadi ke server eksternal**. Semua data disimpan secara lokal di perangkat Anda.

### Data yang Anda berikan
- **Data tubuh**: tinggi, berat, jenis kelamin (digunakan untuk menghitung target hidrasi harian)
- **Preferensi minuman**: nama, ikon, dan warna minuman kustom yang Anda buat
- **Catatan minum**: jumlah, jenis minuman, waktu pencatatan

### Apple Health (HealthKit)

Dengan izin eksplisit Anda, WaterVault dapat menulis catatan minum dan asupan kafein ke Apple Health. Anda dapat mengelola atau mencabut akses HealthKit kapan saja di iOS: «Pengaturan → Kesehatan → Akses data & perangkat».

**Aplikasi ini tidak akan pernah menggunakan data HealthKit untuk iklan atau analitik, dan tidak membagikannya dengan pihak ketiga.**

### Notifikasi pengingat

Jika Anda mengaktifkan pengingat hidrasi, aplikasi akan meminta izin untuk notifikasi lokal. Konten notifikasi dibuat di perangkat dan tidak melibatkan transmisi jaringan.

### Pembelian dalam aplikasi

Pembelian Premium diproses melalui StoreKit 2 Apple. Kami tidak memiliki akses ke informasi pembayaran Anda.

### Sinkronisasi Apple Watch

Jika Anda menggunakan iPhone dan Apple Watch, catatan minum ditransfer langsung antar perangkat melalui kerangka WatchConnectivity Apple, tanpa melalui server eksternal.

## Berbagi data

**Aplikasi ini tidak membagikan data Anda dengan pihak ketiga.**

- Tanpa pengumpulan data analitik
- Tanpa SDK iklan
- Tanpa pendaftaran akun pengguna
- Tanpa unggahan data ke server eksternal
- Tanpa pelacakan perilaku penggunaan

## Penghapusan data

Anda dapat menghapus semua data dengan cara berikut:
1. «Pengaturan → Data → Hapus semua data»
2. Mencopot pemasangan aplikasi
3. Menghapus entri WaterVault di aplikasi Kesehatan

## Layanan pihak ketiga

Satu-satunya layanan pihak ketiga yang digunakan oleh aplikasi ini adalah milik Apple:
- **StoreKit** (pemrosesan pembelian dalam aplikasi)
- **HealthKit** (integrasi data kesehatan)

Layanan-layanan ini tunduk pada kebijakan privasi Apple sendiri.
