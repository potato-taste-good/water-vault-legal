# Política de Privacidade do Drople

**Última atualização: 21 de julho de 2026**

Obrigado por usar o Drople. Esta política de privacidade explica como lidamos com suas informações.

## Coleta e uso de informações

Este aplicativo **não coleta nem envia dados pessoais para servidores externos**. Todos os dados são armazenados localmente no seu dispositivo.

### Dados que você fornece
- **Dados corporais**: altura, peso, sexo (usados para calcular sua meta diária de hidratação)
- **Preferências de bebidas**: nomes, ícones e cores das bebidas personalizadas que você criar
- **Registros de consumo**: quantidade, tipo de bebida, horário do registro

### Apple Health (HealthKit)

Com sua autorização explícita, o Drople pode escrever registros de consumo de água e cafeína no Apple Health. Você pode gerenciar ou revogar o acesso ao HealthKit a qualquer momento no iOS: «Ajustes → Saúde → Acesso a dados e dispositivos».

**Este aplicativo nunca usará dados do HealthKit para publicidade ou análise, nem os compartilhará com terceiros.**

### Notificações de lembrete

Se você ativar os lembretes de hidratação, o aplicativo solicitará permissão para enviar notificações locais. O conteúdo das notificações é gerado no dispositivo e não envolve transmissão em rede.

### Compras no aplicativo

As compras Premium são processadas através do StoreKit 2 da Apple. Não temos acesso às suas informações de pagamento.

### Sincronização com Apple Watch

Se você usa iPhone e Apple Watch, os registros de consumo são transferidos diretamente entre os dispositivos através do framework WatchConnectivity da Apple, sem passar por servidores externos.

## Compartilhamento de dados

**Este aplicativo não compartilha seus dados com terceiros.**

- Sem coleta de dados analíticos
- Sem SDK de publicidade
- Sem registro de conta de usuário
- Sem envio de dados para servidores externos
- Sem rastreamento do seu comportamento de uso

## Exclusão de dados

Você pode excluir todos os seus dados das seguintes formas:
1. «Ajustes → Dados → Limpar todos os dados»
2. Desinstalando o aplicativo
3. Excluindo as entradas do Drople no aplicativo Saúde

## Serviços de terceiros

Os únicos serviços de terceiros utilizados por este aplicativo são da Apple:
- **StoreKit** (processamento de compras no aplicativo)
- **HealthKit** (integração de dados de saúde)

Estes serviços estão sujeitos à política de privacidade da Apple.
