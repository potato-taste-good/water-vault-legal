# Politique de Confidentialité de WaterVault

**Dernière mise à jour : 21 juillet 2026**

Merci d'utiliser WaterVault. Cette politique de confidentialité explique comment nous traitons vos informations.

## Collecte et utilisation des informations

Cette application **ne collecte ni ne télécharge aucune donnée personnelle vers des serveurs externes**. Toutes les données sont stockées localement sur votre appareil.

### Données que vous fournissez
- **Données corporelles** : taille, poids, sexe (utilisées pour calculer votre objectif d'hydratation quotidien)
- **Préférences de boissons** : noms, icônes et couleurs des boissons personnalisées que vous créez
- **Enregistrements de consommation** : quantité, type de boisson, heure d'enregistrement

### Apple Health (HealthKit)

Avec votre autorisation explicite, WaterVault peut écrire les enregistrements de consommation d'eau et de caféine dans Apple Health. Vous pouvez gérer ou révoquer l'accès à HealthKit à tout moment dans les Réglages iOS → Santé → Accès aux données et appareils.

**Cette application n'utilisera jamais les données HealthKit à des fins publicitaires ou d'analyse, et ne les partagera pas avec des tiers.**

### Notifications de rappel

Si vous activez les rappels d'hydratation, l'application demandera l'autorisation d'envoyer des notifications locales. Le contenu des notifications est généré sur l'appareil et ne fait pas l'objet d'une transmission réseau.

### Achats intégrés

Les achats Premium sont traités via StoreKit 2 d'Apple. Nous n'avons pas accès à vos informations de paiement.

### Synchronisation Apple Watch

Si vous utilisez à la fois un iPhone et une Apple Watch, les enregistrements de consommation sont transférés directement entre les appareils via le framework WatchConnectivity d'Apple, sans passer par des serveurs externes.

## Partage des données

**Cette application ne partage pas vos données avec des tiers.**

- Pas de collecte de données analytiques
- Pas de SDK publicitaire
- Pas d'inscription de compte utilisateur
- Pas de téléchargement de données vers des serveurs externes
- Pas de suivi de votre comportement d'utilisation

## Suppression des données

Vous pouvez supprimer toutes vos données en :
1. Accédant à « Réglages → Données → Effacer toutes les données »
2. Désinstallant l'application
3. Supprimant les entrées WaterVault dans l'application Santé

## Services tiers

Les seuls services tiers utilisés par cette application sont ceux d'Apple :
- **StoreKit** (traitement des achats intégrés)
- **HealthKit** (intégration des données de santé)

Ces services sont soumis à la politique de confidentialité d'Apple.
