# WaterVault 개인정보 처리방침

**최종 업데이트: 2026년 7월 21일**

WaterVault를 이용해 주셔서 감사합니다. 본 개인정보 처리방침은 당사가 귀하의 정보를 처리하는 방법을 설명합니다.

## 정보 수집 및 이용

본 앱은**개인 데이터를 외부 서버에 수집 또는 업로드하지 않습니다**. 모든 데이터는 귀하의 기기에 로컬로 저장됩니다.

### 귀하가 제공하는 데이터
- **신체 데이터**: 키, 몸무게, 성별 (일일 수분 목표 계산에 사용)
- **음료 설정**: 귀하가 생성한 맞춤 음료 이름, 아이콘, 색상
- **음수 기록**: 음수량, 음료 유형, 기록 시간

### Apple Health(HealthKit)

귀하의 명시적 동의를 얻어 WaterVault는 음수 기록과 카페인 섭취를 Apple Health에 쓸 수 있습니다. iOS의 「설정 → 건강 → 데이터 접근 및 기기」에서 언제든지 HealthKit 접근 권한을 관리하거나 철회할 수 있습니다.

**본 앱은 HealthKit 데이터를 광고나 분석에 사용하거나 제3자와 공유하지 않습니다.**

### 알림

수분 알림을 활성화하면 로컬 알림 권한을 요청합니다. 알림 내용은 기기에서 생성되며 네트워크 전송이 이루어지지 않습니다.

### 인앱 구매

프리미엄 기능 구매는 Apple의 StoreKit 2를 통해 처리됩니다. 당사는 귀하의 결제 정보에 접근할 수 없습니다.

### Apple Watch 동기화

iPhone과 Apple Watch를 모두 사용하는 경우, 음수 기록은 Apple의 WatchConnectivity 프레임워크를 통해 기기 간 직접 전송되며 외부 서버를 거치지 않습니다.

## 데이터 공유

**본 앱은 귀하의 데이터를 제3자와 공유하지 않습니다.**

- 분석 데이터 수집 없음
- 광고 SDK 없음
- 사용자 계정 등록 없음
- 외부 서버로 데이터 업로드 없음
- 사용 행동 추적 없음

## 데이터 삭제

다음 방법으로 모든 데이터를 삭제할 수 있습니다:
1. 「설정 → 데이터 → 모든 데이터 지우기」
2. 앱 제거
3. 「건강」앱에서 WaterVault 데이터 삭제

## 제3자 서비스

본 앱이 사용하는 제3자 서비스는 Apple만 해당합니다:
- **StoreKit**(인앱 구매 처리)
- **HealthKit**(건강 데이터 통합)

이러한 서비스는 각각 Apple의 개인정보 처리방침을 따릅니다.
